# Pasaporte Stanley

Web estática de campaña para compradores Stanley Bolivia. La experiencia permite registrar una compra válida, abrir un pasaporte digital, completar 12 misiones de Instagram, cargar evidencias y desbloquear sellos.

## Archivos principales

```
index.html      landing + registro
jugar.html      Mi Pasaporte: misiones, sellos y progreso
ranking.html    Comunidad: niveles, beneficios y galería
etapa2.html     Misiones especiales
styles.css      estilos base de campaña
jugar.css       componentes internos del pasaporte
app.js          registro + carrusel
pasaporte.js    lógica local de misiones, evidencias y niveles
config.js       configuración pública de campaña
```

## Mecánica

- 1 compra válida = 1 inscripción = 1 persona participante.
- El participante publica una misión en Instagram, etiqueta a @Stanley1913_Bolivia, toma captura y la sube a la web.
- Cada evidencia cargada desbloquea un sello.
- Bronze: 4 sellos.
- Silver: 7 sellos y participación por premios intermedios.
- Gold: 10 sellos y participación en el sorteo general de Pasaporte Stanley.
- Legend: 12 sellos y participación en el sorteo del Kit especial Stanley.

Pasaporte Stanley es una dinámica recreativa y promocional. No constituye apuesta, juego de azar, actividad de gambling, servicio de pronósticos deportivos pagados ni plataforma de apuestas.
